Menu item service tests

There is a functional test to ensure you can successfully add a menu item

The positive functional test tests to update availability of an item when it is correctly attached to a menu item and restaurant.

There is another functional test to ensure you can edit and remove menu items.

There are two edge case tests that test the scenarios if a restaurant doesnt exist and if a menu item doesnt exist.

There is one last edge case to ensure an unrelated owner cannot edit others menu
